Available resources

These are the resources available for your use:

  • https://holoapi.com/v1/vtuber: Shows a full list of available vtubers in the API Rest
  • https://holoapi.com/v1/vtuber/{id} Shows a vtuber information searching by id
  • https://holoapi.com/v1/vtuber/find? Shows a vtuber information searching by params: fullname, kanji, fanname or youtube or both.
  • https://holoapi.com/v1/vtuber/random Shows a vtuber information by random searching
  • https://holoapi.com/v1/songs Shows a full list of available songs of vtubers
  • https://holoapi.com/v1/songs/{id} Shows a song searching by id
  • https://holoapi.com/v1/aliases: Shows a full list of available aliases of vtubers in the API Rest
  • https://holoapi.com/v1/aliases/{id} Shows an alias searching by id
Note: Only resources for get requests are shown. Post, delete, and put and patch requests are blocked for security.
For more information, see the docs section. you can also test the available resources.
